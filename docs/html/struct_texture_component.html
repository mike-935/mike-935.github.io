Detailed Description

The TextureComponent class is a component that allows us to render textures for our game objects.

This class is a component that allows us to render textures to the screen. It will have a transform component and a collision component which it uses to know where to render, the width and height, as well as the angle.

◆ Render()

void TextureComponent::Render ( )
overridevirtual

Render the texture component.

This function will render the texture component to the screen. All game objects have a transform component so we will always have an x and y to render to. But since collision2DComponent is optional, we assume the width and height are 1 if it is not set.
