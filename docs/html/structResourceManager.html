Detailed Description

The ResourceManager struct is a singleton that holds all of the resources that are loaded in the game.

This struct is a singleton that holds all of the resources that are loaded in the game. For now it just used for handling our textures.

Load a texture from a file path.

This function will load a texture from a file path and return it. If the texture is already loaded it will return the texture that is already loaded. This is how we avoid reloading the same texture multiple times.
