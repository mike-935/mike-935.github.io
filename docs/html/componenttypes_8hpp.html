enum class ComponentType : short
strong

All the current supported components.

This is an enum class that holds all the current supported components.

TextureComponent: A component that allows a gameobject to have a texture and be rendered Collision2DComponent: A component that allows a gameobject to have 2D collisions (holds their w,h and if they are a trigger). TransformComponent: A component that allows a gameobject to have a position and angle. InputComponent: A component that allows a gameobject to track player input. AudioComponent: A component that allows a gameobject to play audio.
